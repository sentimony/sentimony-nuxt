## Навіщо

README функціональний, але не має стислого візуального представлення tech stack
і проєкту в стилі референсу AgileCharts.

## Очікуваний результат

README швидко пояснює технології та ключові workflows через стримані badges/icons,
не дублюючи AGENTS або operational documentation.

## Обсяг

- Додати компактний badge row і логічні section icons.
- Зберегти читабельність без зовнішніх декоративних залежностей.
- Перевірити всі посилання та alt text.

## Залежності

- Узгоджений візуальний стиль і актуальний tech stack.

## Критерії завершення

- README коректно рендериться на GitHub.
- Badges відповідають фактичним версіям/workflows.
- Operational правила не дублюються.

## Наступний крок

Вибрати мінімальний набір badges і section icons за brand reference.
